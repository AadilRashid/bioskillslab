Chapter 3: Classification, Regression and Clustering

~20 min Beginner Classification, Regression, Clustering, Supervised, Unsupervised

The Biological Question

Before choosing any algorithm, answer one question: what kind of output do you want?

Three scenarios from real biology:

  • Which cancer subtype is this tumour? A category
  • How long will this patient survive in months? A number
  • Do these 10,000 cells form distinct natural groups? Unknown groups

These map to the three fundamental ML tasks: classification, regression, and clustering.

Classification — Predicting a Category

Classification assigns an input to one of a fixed set of categories.

Gene expression
→ Cancer subtype
Cell gene expression
→ Cell type (T cell / B cell)
DNA sequence
→ Regulatory element (yes / no)
Variant
→ Pathogenic / Benign

Two categories = binary classification. Three or more = multi-class classification.

Regression — Predicting a Number

Regression predicts a continuous numerical value rather than a category.

Gene expression profile
→ Survival time (months)
Multi-omics profile
→ Drug response score (IC50)
DNA sequence
→ Gene expression level
Cell state features
→ Pseudotime value
Common mistake

Binning survival time into "short" vs "long" survival makes it classification — but discards information and reduces statistical power. Use regression when your outcome is continuous.

Clustering — Discovering Groups Without Labels

Clustering finds natural groupings in data without being told what the groups are. No labels needed.

10,000 single cells
→ 8 cell type clusters
500 tumour samples
→ 4 molecular subtypes
1,000 patients
→ 3 stratification groups
Clusters are not guaranteed to be biology

The groups could reflect batch effects, sequencing depth, or technical variation. Always validate clusters against known markers before making biological claims.

The Three Tasks at a Glance

Task Output Labels? Example
Classification Category Yes Cancer subtype, cell type
Regression Number Yes Survival time, expression level
Clustering Groups No Cell populations, patient groups

Before Building Any Model — Ask These First

  1. What is my output? Category = classification. Number = regression. Unknown groups = clustering.
  2. Do I have labels? Yes = supervised. No = unsupervised.
  3. Is a simpler method sufficient? A t-test, PCA, or differential expression analysis might answer your question without a model at all.
Reality Check
  • Does your question actually require machine learning?
  • If you found clusters, have you validated them against known biology?
  • If you are classifying, where did your labels come from?
