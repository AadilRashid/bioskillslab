Project 4: Single-Cell RNA-Seq — Mapping Human Blood Cells

~3 hours Intermediate scRNA-Seq, Seurat, Scanpy, UMAP, Cell Types, PBMC

The Biological Question

A blood sample contains dozens of different cell types — T cells, B cells, NK cells, monocytes, dendritic cells. Traditional bulk RNA-Seq averages expression across all of them, masking the differences. Single-cell RNA-Seq lets you measure gene expression in each individual cell and figure out what type it is.

In this project you'll take raw 10x Genomics data from 2,700 human peripheral blood mononuclear cells (PBMCs), process it from scratch, cluster the cells, and annotate each cluster with a cell type — all using real published data.

Why PBMC 3k?

The PBMC 3k dataset from 10x Genomics is the "Hello World" of single-cell analysis. It's small enough to run on a laptop, well-characterized so you can verify your results, and used in virtually every scRNA-Seq tutorial worldwide. If you can do this, you can do any scRNA-Seq project.

Step 2: Quality Control

Not all cells are good. Dead cells leak cytoplasmic RNA, leaving mostly mitochondrial transcripts. Doublets (two cells captured together) have unusually high gene counts. You need to filter these out.

Python (Scanpy)

# Calculate QC metrics
# Mitochondrial genes start with "MT-"
adata.var['mt'] = adata.var_names.str.startswith('MT-')
sc.pp.calculate_qc_metrics(adata, qc_vars=['mt'], percent_top=None,
                            log1p=False, inplace=True)

# Visualize QC metrics
sc.pl.violin(adata,
             ['n_genes_by_counts', 'total_counts', 'pct_counts_mt'],
             jitter=0.4, multi_panel=True)

# Filter cells:
# - At least 200 genes detected
# - No more than 2500 genes (doublet filter)
# - Less than 5% mitochondrial reads (dead cell filter)
sc.pp.filter_cells(adata, min_genes=200)
sc.pp.filter_genes(adata, min_cells=3)
adata = adata[adata.obs.n_genes_by_counts < 2500, :]
adata = adata[adata.obs.pct_counts_mt < 5, :]

print(f"Cells after QC: {adata.n_obs}")
print(f"Genes after QC: {adata.n_vars}")

R (Seurat)

# Calculate % mitochondrial reads
pbmc[['percent.mt']] <- PercentageFeatureSet(pbmc, pattern = '^MT-')

# Visualize QC metrics
VlnPlot(pbmc, features = c('nFeature_RNA', 'nCount_RNA', 'percent.mt'),
        ncol = 3)

# Scatter plots to spot outliers
plot1 <- FeatureScatter(pbmc, feature1='nCount_RNA', feature2='percent.mt')
plot2 <- FeatureScatter(pbmc, feature1='nCount_RNA', feature2='nFeature_RNA')
plot1 + plot2

# Filter: 200-2500 genes, <5% mitochondrial
pbmc <- subset(pbmc,
               subset = nFeature_RNA > 200 &
                        nFeature_RNA < 2500 &
                        percent.mt < 5)
pbmc
# Should have ~2638 cells remaining
QC thresholds are dataset-specific

The 5% mitochondrial cutoff and 2500 gene maximum are reasonable for PBMCs but will differ for other tissues. Always look at the distribution first, then set thresholds based on what you see — not arbitrary numbers from a tutorial.

Step 3: Normalization and Feature Selection

Each cell is sequenced to a different depth. Normalization makes cells comparable. Then you select the most variable genes — these carry the most information about cell type differences.

Python (Scanpy)

# Normalize: scale each cell to 10,000 total counts, then log-transform
sc.pp.normalize_total(adata, target_sum=1e4)
sc.pp.log1p(adata)

# Save raw normalized counts for later (needed for DE analysis)
adata.raw = adata

# Find highly variable genes (top 2000)
sc.pp.highly_variable_genes(adata, min_mean=0.0125, max_mean=3,
                             min_disp=0.5)
print(f"Highly variable genes: {adata.var.highly_variable.sum()}")

# Plot variable genes
sc.pl.highly_variable_genes(adata)

# Keep only highly variable genes for downstream analysis
adata = adata[:, adata.var.highly_variable]

# Scale data (zero mean, unit variance) — needed for PCA
sc.pp.scale(adata, max_value=10)

R (Seurat)

# Normalize: log-normalize to 10,000 counts per cell
pbmc <- NormalizeData(pbmc, normalization.method='LogNormalize',
                      scale.factor=10000)

# Find top 2000 variable features
pbmc <- FindVariableFeatures(pbmc, selection.method='vst',
                              nfeatures=2000)

# Plot top variable genes
top10 <- head(VariableFeatures(pbmc), 10)
plot1 <- VariableFeaturePlot(pbmc)
plot2 <- LabelPoints(plot=plot1, points=top10, repel=TRUE)
plot2

# Scale data
all.genes <- rownames(pbmc)
pbmc <- ScaleData(pbmc, features=all.genes)

Step 4: Dimensionality Reduction — PCA

You have ~2000 genes per cell. That's 2000 dimensions. PCA compresses this into ~50 meaningful dimensions by finding the directions of maximum variance. This makes clustering much faster and more reliable.

Python (Scanpy)

# Run PCA
sc.tl.pca(adata, svd_solver='arpack')

# Plot variance explained by each PC
sc.pl.pca_variance_ratio(adata, log=True, n_pcs=50)
# Look for the "elbow" — where adding more PCs stops helping

# Visualize cells in PCA space
sc.pl.pca(adata, color='CST3')  # CST3 = monocyte marker

R (Seurat)

# Run PCA
pbmc <- RunPCA(pbmc, features=VariableFeatures(object=pbmc))

# Visualize top genes driving each PC
VizDimLoadings(pbmc, dims=1:2, reduction='pca')

# Elbow plot to choose number of PCs
ElbowPlot(pbmc)
# The elbow is usually around PC 10-15 for PBMCs

Step 5: Clustering

Now group similar cells together. The algorithm builds a graph where cells are connected to their nearest neighbors, then finds communities in that graph. Each community is a cluster — likely a distinct cell type.

Python (Scanpy)

# Build neighborhood graph using top 10 PCs
sc.pp.neighbors(adata, n_neighbors=10, n_pcs=40)

# Cluster using Leiden algorithm (better than Louvain)
sc.tl.leiden(adata, resolution=0.5)
print(f"Number of clusters: {adata.obs['leiden'].nunique()}")

# Run UMAP for visualization
sc.tl.umap(adata)

# Plot UMAP colored by cluster
sc.pl.umap(adata, color=['leiden'], legend_loc='on data',
           title='PBMC clusters', frameon=False)

R (Seurat)

# Build KNN graph using top 10 PCs
pbmc <- FindNeighbors(pbmc, dims=1:10)

# Cluster (resolution controls granularity: higher = more clusters)
pbmc <- FindClusters(pbmc, resolution=0.5)
print(paste("Number of clusters:", length(levels(pbmc))))

# Run UMAP
pbmc <- RunUMAP(pbmc, dims=1:10)

# Plot
DimPlot(pbmc, reduction='umap', label=TRUE) + NoLegend()
How many clusters?

For PBMC 3k at resolution 0.5, you should get around 8-9 clusters. If you get too many, lower the resolution. Too few, raise it. There's no single "correct" number — it depends on the biology you care about.

Step 6: Find Marker Genes

Each cluster should have genes that are uniquely highly expressed compared to all other clusters. These marker genes tell you what cell type each cluster is.

Python (Scanpy)

# Find marker genes for each cluster
sc.tl.rank_genes_groups(adata, 'leiden', method='wilcoxon')
sc.pl.rank_genes_groups(adata, n_genes=25, sharey=False)

# Get top markers as a dataframe
markers = sc.get.rank_genes_groups_df(adata, group=None)
print(markers.head(20))

# Dot plot of top markers per cluster
sc.pl.rank_genes_groups_dotplot(adata, n_genes=5)

R (Seurat)

# Find markers for every cluster vs all others
pbmc.markers <- FindAllMarkers(pbmc,
                                only.pos=TRUE,
                                min.pct=0.25,
                                logfc.threshold=0.25)

# Top 2 markers per cluster
pbmc.markers %>%
  group_by(cluster) %>%
  slice_max(n=2, order_by=avg_log2FC)

# Heatmap of top 10 markers per cluster
top10 <- pbmc.markers %>%
  group_by(cluster) %>%
  slice_max(n=10, order_by=avg_log2FC)
DoHeatmap(pbmc, features=top10$gene) + NoLegend()

Step 7: Cell Type Annotation

Match your marker genes to known cell type markers. This is where biology knowledge matters.

# Known PBMC marker genes:
# CD3D, CD3E, CD3G  → T cells (all T cells)
# CD4               → CD4+ T cells (helper T cells)
# CD8A, CD8B        → CD8+ T cells (cytotoxic T cells)
# MS4A1, CD79A      → B cells
# GNLY, NKG7        → NK cells
# CD14, LYZ         → CD14+ Monocytes
# FCGR3A, MS4A7     → CD16+ Monocytes
# FCER1A, CST3      → Dendritic cells
# PPBP              → Megakaryocytes (platelets)

Python (Scanpy)

# Visualize known markers on UMAP
marker_genes = ['CD3D', 'CD4', 'CD8A', 'MS4A1', 'GNLY',
                'CD14', 'LYZ', 'FCGR3A', 'FCER1A', 'PPBP']
sc.pl.umap(adata, color=marker_genes, ncols=5)

# Assign cell type labels to clusters
# (adjust based on YOUR cluster numbers)
cell_type_map = {
    '0': 'CD4+ T cells',
    '1': 'CD14+ Monocytes',
    '2': 'CD4+ T cells',
    '3': 'B cells',
    '4': 'CD8+ T cells',
    '5': 'NK cells',
    '6': 'CD14+ Monocytes',
    '7': 'Dendritic cells',
    '8': 'CD16+ Monocytes',
}
adata.obs['cell_type'] = adata.obs['leiden'].map(cell_type_map)

# Final annotated UMAP
sc.pl.umap(adata, color='cell_type', legend_loc='on data',
           title='PBMC Cell Types', frameon=False,
           palette='tab10')

R (Seurat)

# Visualize known markers
FeaturePlot(pbmc,
            features=c('CD3D','CD4','CD8A','MS4A1',
                       'GNLY','CD14','LYZ','FCGR3A'),

# Assign cell type labels (adjust cluster numbers to match yours)
new.cluster.ids <- c('Naive CD4 T', 'CD14+ Mono', 'Memory CD4 T',

names(new.cluster.ids) <- levels(pbmc)

# Final annotated UMAP

Challenge Extensions

  1. Try different resolutions — Run clustering at 0.3, 0.5, 0.8 and compare the UMAPs. How does resolution affect cell type granularity?
  2. Automated annotation — Use celltypist (Python) or SingleR (R) to automatically annotate cell types without manual marker lookup
  3. Trajectory analysis — Use scVelo (Python) to infer RNA velocity and see which cells are differentiating into which
  4. Integrate two datasets — Download a second PBMC dataset and integrate them using Harmony or Seurat's integration workflow to remove batch effects
  5. Doublet detection — Run scrublet (Python) or DoubletFinder (R) to computationally identify doublets before filtering
