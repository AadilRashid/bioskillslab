Chapter 1: What is Bioinformatics?

~40 min Beginner Biology, Data, History, Genomics

What is Bioinformatics?

Bioinformatics is the science of collecting, storing, analyzing, and interpreting biological data using computational methods. It emerged from the collision of two revolutions: the molecular biology revolution (understanding DNA, RNA, and proteins) and the computing revolution (cheap, fast processors and massive storage).

Modern bioinformatics encompasses a wide range of activities:

  • Genome assembly — Piecing together millions of short DNA reads into a complete genome
  • Gene prediction & annotation — Finding genes in a genome and assigning functions
  • Sequence alignment — Comparing sequences to find similarities and differences
  • Phylogenetics — Reconstructing evolutionary relationships between organisms
  • Structural biology — Predicting 3D protein structures (e.g., AlphaFold)
  • Systems biology — Modeling entire biological networks and pathways
  • Clinical genomics — Using genomic data for diagnosis and treatment decisions

The Central Dogma of Molecular Biology

Before you can analyze biological data, you need to understand what that data represents. The central dogma describes the flow of genetic information in cells:

        Replication
        ┌───────┐
        ▼       │
       DNA ────────→ RNA ────────→ Protein
           Transcription    Translation

DNA — The Blueprint

DNA (deoxyribonucleic acid) is a double-stranded molecule made of four nucleotide bases:

  • A (Adenine) pairs with T (Thymine)
  • C (Cytosine) pairs with G (Guanine)

The two strands run in opposite directions (5'→3' and 3'→5'). This is called antiparallel orientation and is critical for understanding sequencing data.

5' ─── A T C G G C T A A T ─── 3'   (forward strand)
       | | | | | | | | | |
3' ─── T A G C C G A T T A ─── 5'   (reverse strand / reverse complement)

RNA — The Working Copy

RNA is single-stranded and uses Uracil (U) instead of Thymine (T). Messenger RNA (mRNA) carries the genetic code from DNA to ribosomes. Other types include:

  • mRNA — Carries protein-coding information
  • tRNA — Transfers amino acids during translation
  • rRNA — Structural component of ribosomes
  • miRNA, lncRNA, etc. — Regulatory non-coding RNAs (a hot research area)

Protein — The Machinery

Proteins are chains of amino acids encoded by triplets of nucleotides called codons. There are 20 standard amino acids, and the genetic code maps 64 possible codons to these 20 amino acids (plus stop signals).

DNA:     ATG  GCC  ATT  GTA  ATG  TAG
mRNA:    AUG  GCC  AUU  GUA  AUG  UAG
Protein: Met  Ala  Ile  Val  Met  STOP
Key Numbers to Remember
  • Human genome: ~3.2 billion base pairs
  • ~20,000 protein-coding genes (only ~1.5% of the genome)
  • ~98.5% is non-coding (regulatory regions, introns, repetitive elements, etc.)
  • Any two humans share ~99.9% of their DNA

The Sequencing Revolution

The ability to read DNA sequences has transformed biology. Here's how the technology evolved:

First Generation — Sanger Sequencing (1977)

Developed by Frederick Sanger. Reads ~800 bp at a time. Used to sequence the first human genome (Human Genome Project, 1990–2003, $3 billion).

Second Generation — Next-Generation Sequencing (2005+)

Illumina dominates this space. Produces millions of short reads (75–300 bp) in parallel. A human genome now costs under $200 and takes ~24 hours.

Third Generation — Long-Read Sequencing

  • PacBio HiFi — Reads of 10–25 kb with high accuracy (~99.9%)
  • Oxford Nanopore — Reads of 10 kb to >1 Mb, portable (MinION fits in your hand)

Long reads are revolutionizing genome assembly, structural variant detection, and direct RNA sequencing.

The Data Explosion

The NCBI Sequence Read Archive (SRA) stores over 70 petabytes of sequencing data and doubles roughly every 18 months. A single RNA-Seq experiment can produce 50–200 GB of raw data. This is why bioinformatics skills are essential — you simply cannot analyze this data by hand.

A Typical Bioinformatics Workflow

Most genomics analyses follow a similar pattern. Understanding this pipeline is the backbone of this entire course:

┌─────────────────────────────────────────────────────┐
│  1. EXPERIMENTAL DESIGN                              │
│     Define question, controls, replicates            │
└──────────────────────┬──────────────────────────────┘
                       ▼
┌─────────────────────────────────────────────────────┐
│  2. SEQUENCING                                       │
│     Illumina / PacBio / Nanopore → Raw FASTQ files   │
└──────────────────────┬──────────────────────────────┘
                       ▼
┌─────────────────────────────────────────────────────┐
│  3. QUALITY CONTROL                                  │
│     FastQC → assess quality                          │
│     fastp/Trimmomatic → trim adapters, low-quality   │
└──────────────────────┬──────────────────────────────┘
                       ▼
┌─────────────────────────────────────────────────────┐
│  4. ALIGNMENT                                        │
│     BWA/STAR/Bowtie2 → map reads to reference        │
│     Output: SAM/BAM files                            │
└──────────────────────┬──────────────────────────────┘
                       ▼
┌─────────────────────────────────────────────────────┐
│  5. POST-PROCESSING                                  │
│     samtools sort/index, mark duplicates             │
└──────────────────────┬──────────────────────────────┘
                       ▼
┌─────────────────────────────────────────────────────┐
│  6. ANALYSIS (depends on experiment type)             │

└─────────────────────────────────────────────────────┘

Reproducibility — The Golden Rule

In bioinformatics, if it's not reproducible, it's not science. A 2016 Nature survey found that 70% of researchers failed to reproduce another scientist's experiments. Computational work should be easier to reproduce than wet-lab work — but only if you follow good practices.

The Reproducibility Checklist

  • Version control everything — Use Git for all code and scripts
  • Document your environment — Conda environment.yml or Docker containers
  • Automate your pipeline — Snakemake, Nextflow, or at minimum a Makefile
  • Never modify raw data — Treat original files as read-only
  • Script your figures — Every plot should be generated by code, not manual editing
  • Record software versions — samtools --version, R --version, etc.
  • Use checksums — Verify data integrity with md5sum/sha256sum
The Cardinal Sin of Bioinformatics

Never modify your raw data files. Never open a FASTQ in Excel and "fix" something. Always write processed results to new files and keep a clear trail from raw → processed → results.

Key Databases You'll Use Throughout This Course

  • NCBI — The US hub. GenBank (sequences), PubMed (literature), SRA (raw reads), GEO (expression data), dbSNP (variants)
  • Ensembl — European genome annotation. Best for downloading reference genomes and GTF files
  • UniProt — Protein sequences and functional annotation. Swiss-Prot (curated) vs TrEMBL (automated)
  • UCSC Genome Browser — Visual genome exploration with hundreds of annotation tracks
  • PDB — Protein Data Bank. 3D structures of proteins
  • KEGG / Reactome — Biological pathway databases
