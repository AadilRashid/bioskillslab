Chapter 10: Correlation and Linear Regression

~45 min Intermediate Pearson, Spearman, Regression, R², Residuals

Correlation: Do Two Things Move Together?

Correlation measures the strength and direction of a linear relationship between two variables. It doesn't tell you why they're related, just that they are.

Pearson correlation (r)

Measures linear association. Ranges from -1 to +1.

r = +1.0   Perfect positive linear relationship
r = +0.7   Strong positive
r = +0.3   Weak positive
r =  0.0   No linear relationship
r = -0.3   Weak negative
r = -0.7   Strong negative
r = -1.0   Perfect negative linear relationship
# In R:
cor(x, y)                          # Pearson r
cor.test(x, y)                     # r with p-value and CI
cor(x, y, method = "spearman")     # Spearman rank correlation

Spearman correlation (ρ)

Uses ranks instead of raw values. Measures monotonic (not just linear) relationships. More robust to outliers and non-Gaussian data.

Use Spearman when:

  • Data is skewed or has outliers
  • The relationship is monotonic but not linear (e.g., logarithmic)
  • Data is ordinal (rankings, scores)

Correlation in bioinformatics

  • Sample-sample correlation: Do replicates agree? Pearson r > 0.95 between replicates is typical for RNA-Seq.
  • Gene co-expression: Which genes have correlated expression across samples? (WGCNA networks)
  • QC metric: Does mapping rate correlate with total reads? (It shouldn't.)
Correlation ≠ causation

Two genes being co-expressed doesn't mean one regulates the other. They might both be regulated by a third factor. Or the correlation might be driven by a confounding variable like cell type composition. Always think about what's driving the correlation.

Linear Regression: Predicting One Variable from Another

Correlation tells you the strength of a relationship. Regression gives you the equation.

y = β₀ + β₁x + ε

β₀ = intercept (value of y when x = 0)
β₁ = slope (how much y changes per unit change in x)
ε  = residual error (what the model can't explain)

In R:

model <- lm(expression ~ age, data = df)
summary(model)

# Coefficients:
#             Estimate Std. Error t value Pr(>|t|)
# (Intercept)  120.5      15.2     7.93   0.0001 ***
# age            2.3       0.8     2.88   0.012  *
#
# R-squared: 0.34

# Interpretation: expression increases by 2.3 units per year of age.
# Age explains 34% of the variance in expression.

R² — How Good is the Fit?

R² (coefficient of determination) tells you what fraction of the variance in y is explained by x.

R² = 0.0    x explains nothing about y
R² = 0.3    x explains 30% of the variance in y
R² = 0.7    x explains 70%
R² = 1.0    x perfectly predicts y (never happens with real data)

Note: R² = r² for simple linear regression. A Pearson correlation of 0.7 means R² = 0.49, so x explains about half the variance in y.

Checking Your Regression

A regression line can be misleading. Always check the residuals.

# In R:
par(mfrow = c(2,2))
plot(model)

# What to look for:
# 1. Residuals vs Fitted: should be random scatter, no pattern
# 2. Q-Q plot: residuals should follow the diagonal (normality)
# 3. Scale-Location: spread should be constant (homoscedasticity)
# 4. Residuals vs Leverage: no points with extreme influence

Common problems:

  • Curved pattern in residuals: relationship isn't linear. Try a polynomial or log transform.
  • Fan shape in residuals: variance increases with fitted values. Try log-transforming y.
  • Outliers with high leverage: one point is pulling the line. Investigate that sample.

Regression in Bioinformatics

  • Normalization: Loess regression for MA-plot normalization in RNA-Seq
  • Batch correction: Linear models to regress out batch effects (ComBat, limma)
  • Age/sex correction: Include covariates in the DESeq2 design formula: ~age + sex + condition
  • Standard curves: qPCR quantification uses linear regression on Ct vs log(concentration)
  • Dose-response: Fitting drug response curves (nonlinear regression, Chapter 11)

Correlation vs Regression: When to Use Which

Question                                    Tool
────────                                    ────
"Are these two things related?"             Correlation
"How strong is the relationship?"           Correlation (r)
"Can I predict y from x?"                   Regression
"How much does y change per unit of x?"     Regression (slope)
"How much variance does x explain?"         Regression (R²)
"I want to adjust for confounders"          Multiple regression (Ch 11)
