Chapter 9: ANOVA — Comparing Many Groups

~40 min Intermediate ANOVA, Post-hoc, Tukey, Kruskal-Wallis

Why Not Just Run Multiple t-tests?

You have 4 treatment groups and want to know if any differ. You could run t-tests on every pair: A vs B, A vs C, A vs D, B vs C, B vs D, C vs D. That's 6 tests. At p < 0.05 each, your chance of at least one false positive is about 26%. With 10 groups, it's 90 pairwise tests and you're almost guaranteed false positives.

ANOVA solves this by asking one question first: is there any difference among the groups? Only if the answer is yes do you dig into which specific groups differ.

How ANOVA Works

ANOVA compares two sources of variation:

Between-group variance:  How much do the group means differ from each other?
Within-group variance:   How much do individual values vary within each group?

F = between-group variance / within-group variance

If the groups are truly different, between-group variance will be large relative to within-group variance, giving a large F statistic and a small p-value.

If there's no real difference, both variances are just random noise and F will be close to 1.

In R:

# One-way ANOVA
model <- aov(expression ~ treatment, data = df)
summary(model)

#              Df Sum Sq Mean Sq F value  Pr(>F)
# treatment     3   1250   416.7   8.33  0.0012 **
# Residuals    16    800    50.0

p = 0.0012 tells you at least one group differs. But which one? ANOVA doesn't tell you that. You need post-hoc tests.

Post-hoc Tests: Which Groups Differ?

Tukey's HSD (Honest Significant Difference)

The most common post-hoc test. Compares every pair of groups while controlling for multiple comparisons.

TukeyHSD(model)

#                    diff    lwr    upr    p adj
# B-A               8.5     2.1   14.9    0.007
# C-A               2.0    -4.4    8.4    0.820
# D-A              12.0     5.6   18.4    0.001
# C-B              -6.5   -12.9   -0.1    0.045
# D-B               3.5    -2.9    9.9    0.450
# D-C              10.0     3.6   16.4    0.002

Now you can see: A differs from B and D. C differs from D. B and D are similar. C and A are similar.

Other post-hoc options

Tukey's HSD:     All pairwise comparisons, balanced designs
Dunnett's test:  Compare all groups to one control group
Bonferroni:      Conservative, works for any comparison
Scheffé:         Most conservative, works for any contrast

Two-way ANOVA

When you have two factors. For example: treatment (drug A, drug B, placebo) AND sex (male, female). Two-way ANOVA tests three things:

  1. Does treatment matter? (main effect of treatment)
  2. Does sex matter? (main effect of sex)
  3. Does the treatment effect depend on sex? (interaction)
# In R:
model <- aov(expression ~ treatment * sex, data = df)
summary(model)

# The interaction term (treatment:sex) is often the most interesting.
# If significant, the drug works differently in males vs females.

Assumptions of ANOVA

  • Independence of observations
  • Normality within each group (check with Q-Q plots or Shapiro-Wilk)
  • Homogeneity of variance across groups (check with Levene's test)

ANOVA is fairly robust to mild violations of normality, especially with balanced designs and n > 10 per group. Unequal variances are more problematic.

Kruskal-Wallis Test

The nonparametric alternative to one-way ANOVA. Uses ranks instead of raw values. No normality assumption.

# In R:
kruskal.test(expression ~ treatment, data = df)

# Post-hoc pairwise comparisons:
pairwise.wilcox.test(df$expression, df$treatment, p.adjust.method = "BH")

ANOVA in Bioinformatics

  • Comparing expression across multiple conditions (e.g., 4 time points, 3 drug doses)
  • Batch effect assessment: is there a significant difference in a QC metric across sequencing batches?
  • Cell type comparisons in single-cell: does a gene differ across 5 clusters?
ANOVA vs DESeq2

For RNA-Seq with more than 2 conditions, DESeq2 handles this internally using likelihood ratio tests (LRT). You don't need to run ANOVA on RNA-Seq counts. Use DESeq(dds, test="LRT", reduced=~1) to test if any condition differs.
