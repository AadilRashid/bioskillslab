Statistics Project: Interrogating a Real RNA-Seq Dataset

~3 hours Intermediate All 15 chapters applied to real data

The Question

Dexamethasone is a glucocorticoid used to treat asthma. It works by suppressing inflammation in airway cells. But which genes does it actually affect, and how confident should we be in those findings?

In this project, you'll apply every statistical concept from this course to a real published dataset. You won't just run DESeq2 and call it done. You'll interrogate the results statistically: check distributions, assess power, validate assumptions, interpret effect sizes, and think critically about what the data can and cannot tell you.

The Dataset

Study:    Himes et al., 2014, PLOS ONE (GSE52778)
Samples:  4 untreated + 4 dexamethasone-treated airway smooth muscle cells
Platform: Illumina HiSeq 2000
Access:   Available via Bioconductor airway package

Part 1: Understand Your Data Before Any Testing

Before running a single statistical test, understand what you're working with. This is the most important habit in statistics.

Step 1.1: Check the distribution of counts

# Raw counts are highly skewed (lognormal)
counts_mat <- assay(se)
hist(counts_mat[, 1], breaks = 100, main = "Raw counts — sample 1")
# Most genes have very low counts, a few are very high

# Log-transform to see the distribution better
hist(log2(counts_mat[, 1] + 1), breaks = 50,
     main = "Log2(counts + 1) — sample 1",
     xlab = "Log2 expression")
# Now it looks roughly Gaussian — this is the lognormal distribution

What you should see: raw counts are right-skewed (lognormal). Log-transformed counts are approximately Gaussian. This is why we work on the log scale in genomics.

Step 1.2: Check sample-sample correlations

# Pearson correlation between all pairs of samples
log_counts <- log2(counts_mat + 1)
cor_mat <- cor(log_counts)
pheatmap(cor_mat,
         annotation_col = data.frame(
           treatment = colData(se)$dex,
           row.names = colnames(se)),
         main = "Sample-sample Pearson correlation")

# What to look for:
# - Replicates should correlate > 0.95
# - Treated and untreated should form separate clusters

Step 1.3: PCA for quality control

dds <- DESeqDataSet(se, design = ~dex)
dds <- dds[rowSums(counts(dds)) >= 10, ]
vsd <- vst(dds)

plotPCA(vsd, intgroup = "dex") +
  theme_minimal() +

# 2. Are there any outlier samples?
# 3. What % of variance does PC1 explain?

Part 2: Statistical Testing

Step 2.1: Run DESeq2

dds <- DESeq(dds)
res <- results(dds, contrast = c("dex", "trt", "untrt"), alpha = 0.05)
res <- res[order(res$padj), ]

# Summary
summary(res)
cat("\nGenes tested:", nrow(res))
cat("\nSignificant (padj < 0.05):", sum(!is.na(res$padj) & res$padj < 0.05))
cat("\nWith |log2FC| > 1:", sum(!is.na(res$padj) & res$padj < 0.05 & abs(res$log2FoldChange) > 1))

Step 2.2: Understand the p-value distribution

# Plot histogram of raw p-values
hist(res$pvalue, breaks = 50,
     main = "P-value distribution",
     xlab = "Raw p-value",
     col = "steelblue")

# What to look for:
# - Uniform distribution from 0.1 to 1.0 (null genes)
# - Spike near 0 (truly DE genes)
# - If the distribution is flat or has a spike near 1, something is wrong

A healthy p-value histogram has a spike near 0 (real effects) and a flat distribution from 0.1 to 1.0 (null genes). This is what Benjamini-Hochberg correction assumes.

Step 2.3: Compare raw vs adjusted p-values

# How many genes are significant with raw vs adjusted p-values?
raw_sig <- sum(!is.na(res$pvalue) & res$pvalue < 0.05)
adj_sig <- sum(!is.na(res$padj) & res$padj < 0.05)

cat("Significant with raw p < 0.05:", raw_sig)
cat("Significant with padj < 0.05:", adj_sig)
cat("Difference (likely false positives):", raw_sig - adj_sig)

# This demonstrates why multiple testing correction matters

Part 3: Effect Size Analysis

Step 3.1: Volcano plot

library(EnhancedVolcano)
EnhancedVolcano(as.data.frame(res),
    lab = rownames(res),
    x = "log2FoldChange",
    y = "padj",
    pCutoff = 0.05,
    FCcutoff = 1,
    title = "Dexamethasone vs Untreated",
    subtitle = "Airway smooth muscle cells")

# Identify the quadrants:
# Upper left: downregulated, significant
# Upper right: upregulated, significant
# Lower middle: not significant (regardless of fold change)

Step 3.2: Separate statistical from biological significance

# Genes significant by p-value only (small effect)
stat_only <- subset(res, padj < 0.05 & abs(log2FoldChange) <= 1)
cat("Statistically significant but small effect:", nrow(stat_only))

# Genes significant by both criteria
both_sig <- subset(res, padj < 0.05 & abs(log2FoldChange) > 1)
cat("Both statistically and biologically significant:", nrow(both_sig))

# Distribution of fold changes among significant genes
hist(both_sig$log2FoldChange, breaks = 30,
     main = "Fold changes of significant DE genes",
     xlab = "log2 Fold Change")

Step 3.3: Confidence intervals on fold changes

# DESeq2 provides standard errors for fold changes
# Compute 95% CIs
res_df <- as.data.frame(res)
res_df$CI_lower <- res_df$log2FoldChange - 1.96 * res_df$lfcSE
res_df$CI_upper <- res_df$log2FoldChange + 1.96 * res_df$lfcSE

# Look at the top 10 genes
top10 <- head(res_df[order(res_df$padj), ], 10)
top10[, c("log2FoldChange", "lfcSE", "CI_lower", "CI_upper", "padj")]

# Which genes have tight CIs (precise estimates)?
# Which have wide CIs (uncertain estimates)?

Part 4: Validate Known Biology

# Himes et al. reported CRISPLD2 as a key dex-responsive gene
# Let's check our results

known_genes <- c("CRISPLD2", "DUSP1", "KLF15", "PER1", "TSC22D3")

for (gene in known_genes) {
  idx <- grep(gene, rownames(res))
  if (length(idx) > 0) {
    r <- res[idx[1], ]
    cat(sprintf("\n%s: log2FC = %.2f, padj = %.2e %s",
        gene,
        r$log2FoldChange,
        r$padj,
        ifelse(!is.na(r$padj) & r$padj < 0.05, "SIGNIFICANT", "not significant")))
  }
}

Part 5: Power Analysis

# How many genes could we detect with only 2 replicates instead of 4?
library(RNASeqPower)

# Estimate CV from the data
# CV = SD / mean for each gene
cv_estimates <- apply(counts(dds)[rowSums(counts(dds)) > 100, ], 1, function(x) {
  sd(x) / mean(x)
})
median_cv <- median(cv_estimates, na.rm = TRUE)
cat("Median CV:", round(median_cv, 3))

# Power with n=2 vs n=4 vs n=8
for (n in c(2, 4, 8)) {
  p <- rnapower(depth = 20, n = n, cv = median_cv, effect = 2, alpha = 0.05)
  cat(sprintf("\nn = %d: power = %.2f", n, p))
}
